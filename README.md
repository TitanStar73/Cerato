# Cerato

A python based comprehensive video editing suite. This open-source project aims to be an alternative to extremely expensive proprietary software.

# Installation

1. Clone this repo
2. Move into the `Cerato/` directory
3. Run `pip install requirements.txt`
4. You are all setup! 

# Usage

In the `Cerato/` directory run `python editor.py` to get started!

# Features
- AI Based Upscaler (Can run on [Kaggle](https://www.kaggle.com) so you DON'T needs a powerful GPU!)
- Youtube Downloader
- Integrated scene pack animator (split, crop, rotate, screenshot and add in keyframes to your scenepack)
- Smart Beat Recognition and beat sync
- Automatic foreground background split
- Easy-to-use text creation (automatic aligning, glow, shadows, etc.)
- Integrated clip editor (temporal warps, vibrance, etc.)
- 150+ Filters and Transitions
